# BugTracker

App web para seguimiento de bugs/defectos. Hecha para practicar testing manual y automatizado.

## Funcionalidades
- Login con validacion de campos y credenciales
- Dashboard con estadisticas (total, abiertos, en progreso, cerrados)
- CRUD completo de bugs (crear, ver, editar, eliminar)
- Filtros por estado y prioridad
- Busqueda por texto
- Confirmacion de eliminacion
- Notificaciones toast
- Responsive
